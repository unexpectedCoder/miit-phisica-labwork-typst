# Шаблон отчёта по лабораторной работе кафедры «Физика» РУТ (МИИТ)

Шаблон [Typst](https://typst.app/) отчёта по лабораторным работам по физике для студентов [РУТ (МИИТ)](https://www.miit.ru/).

## Предварительно

1. Ознакомьтесь с Typst через [урок](https://typst.app/docs/tutorial/) по его основам.
2. Установите [VS Code](https://code.visualstudio.com/). Это бесплатный расширяемый текстовый редактор.
    * Установить расширение [Tinymist Typst](https://marketplace.visualstudio.com/items?itemName=myriad-dreamin.tinymist).
    * *При желании установить расширение* [*Typst Sync*](https://marketplace.visualstudio.com/items?itemName=OrangeX4.vscode-typst-sync).
3. Установить в систему шрифты [STIX Two Text](https://fonts.google.com/specimen/STIX+Two+Text) и [STIX Two Math](https://fonts-online.ru/fonts/stix-two-math), используемые в шаблоне (данные шрифты открытые и бесплатные).
    * Скачаются архивы.
    * Откройте их и запустите все найденные файлы с расширением `.ttf`.
    * Откроется установщик шрифтов.
    * Нажмите «Установить».
    * Если VS Code открыт, то закройте его и запустите заново, чтобы он подцепил новые шрифты.

Вместо VS Code можно установить официальное бесплатное приложение typst.app.

## Quickstart

1. Скачайте данный репозиторий как архив.
2. Распакуйте архив в рабочую директорию, где будете работать с проектом отчёта.
3. Откройте рабочую директорию в VS Code (`Файл - Открыть директорию` или `File - Open Folder`) или загрузить файлы в проект в typst.app.
5. Оформить отчёт, редактируя файл [`main.typ`](main.typ). Изначально этот файл содержит:
   * Настройку титульного листа. Здесь нужно указать конкретную информацию, убрав все `highlight` (жёлтое выделение).
   * Некоторые примеры оформления рисунков, таблиц, формул и др. Смело убирайте всё, что вам не нужно.
6. Экспортировать (сохранить) в PDF.
7. Готово!

При оформлении отчёта вся работа сводится к написанию текста внутри основного файла [`main.typ`](main.typ).
Внутри файла приведены некоторые примеры, раскрывающие возможности Typst.

При расширении VS Code для работы с Typst появится возможность предварительного просмотра результата.
Пример итогового файла - [`main.pdf`](main.pdf).

Файл [`labreport.typ`](labreport.typ) содержит все необходимые настройки шаблона документа: стили текста, формул, списков, рисунков, таблиц, ссылок и др.
**Нет необходимости разбираться в этом файле и редактировать его, если вам нужно лишь написать отчёт.**
